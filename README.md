## uniapp + vite + vue3.0 + typescript

* uview-plus
* unocss
* pinia
* pinia-plugin-persistedstate
* vue-i18n
* lyla-uni 请求插件
* z-paging 上拉加载下拉刷新
### 注意

+ pinia 需要固定的版本使用， 才能兼容到 uniapp; 使用 taze 更新依赖时 taze --exclude pinia

+ eslint 规则有可能需要再 setting.json 中配置 `"eslint.experimental.useFlatConfig": true`
